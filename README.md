# Bank Loan & Investment Calculator 🏦

A terminal-based financial calculator built with Python!

## Features
- Loan Calculator — calculates total interest 
  and monthly payment
- Investment Calculator — calculates total 
  returns on investment
- Clean formatted output with ₹ symbol

## How to use
- Choose option 1 for Loan
- Choose option 2 for Investment
- Enter amount, rate, and time
- Get instant results!

## Concepts used
- if-elif conditions
- float input handling
- f-string formatting
- Math calculations
- User menu design
